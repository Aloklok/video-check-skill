## 1、`<Background>` 标签涉及专业术语及样例

### 1.1 场景布局有哪些？（注意结合实际场景观测，以下均为多数场景情况）

| 场景 | 光线类型 |
|------|---------|
| **室外** | |
| 街道/马路 | 自然光、补光、街道灯光 |
| 公园/花园 | 自然光 |
| 海滩 | 自然光、海面反光 |
| 旷野/山顶 | 自然光 |
| 火车站/机场 | 自然光、室内灯光 |
| 夜景街道 | 室外灯光、补光 |
| 雨夜/雪地 | 雪面反光、自然光 |
| **室内** | |
| 办公室 | 室内灯光、室外自然光 |
| 医院/手术室 | 通常为室内冷白灯光、均匀、无阴影 |
| 教室 | 室外自然光、顶光 |
| 餐厅/酒吧 | 彩色灯光、高饱和度光 |
| 家庭客厅 | 温馨自然高饱和度灯光 |
| 卧室 | 多为柔和室内暖光 |
| 审讯室/监狱 | 硬光、顶光 |
| 仓库/工厂 | 冷色调、室内白炽灯 |

| KTV/夜店 | 彩色光、频闪 |
| 汽车内 | LED屏光、车外自然光 |

### 1.2 光线类型及其特点（需区分人造光和自然光）

| 类型 | 特点 | 案例（图片形式描述） |
|------|------|---------------------|
| 逆光 | 日出后或日落前一小时，太阳角度低、光质柔和温暖。用逆光或侧逆光勾勒人物与植物的轮廓，产生金色发丝光、空气透视感，画面极度浪漫治愈。 | |
| 柔光 | 厚厚的云层将直射阳光变成巨大的柔光箱，无阴影，色彩饱和而安静。最适合表现绿色植被的苍翠、雨后湿润的质感，情绪宁静悠远。 | |
| 丁达尔光 | 阳光穿过晨雾、炊烟或树林中的缝隙，形成可见的光柱。为画面注入神圣感与生机感。是田园风光从"好看"升级到"惊艳"的关键。 | |
| 伦勃朗光 | 光线从侧上方斜射而来，照亮主体半边脸或身体，在另一侧脸颊的阴影处形成一个倒三角形的亮区。光比大，暗部深沉但保有细节。 | |
| 顶光 | 正午时分，阳光从头顶直射而下。公认的"死亡光线"，眼窝和下巴都会投下浓重阴影。 | |

## 2、`<Style>` 标签涉及专业术语及样例

### 2.1 常见画风类型，仅供参考

| 画风 | 特点 | 例子 |
|------|------|------|
| 写实风格 | 追求真实感，色彩、光影和场景都尽量贴近现实，注重细节的真实呈现。适合纪录片、现实题材的剧情片等。 | 《舌尖上的中国》（纪录片） |
| 卡通风格 | 色彩鲜艳、线条简洁，角色造型夸张、可爱。适合儿童动画或轻松幽默的视频。 | 《喜羊羊与灰太狼》（儿童动画） |
| 像素风格 | 采用低分辨率的像素点阵构成画面，色彩鲜明且高度概括，刻意营造复古电子游戏的视觉质感。它剥去复杂细节，以极简的方式与轮廓，将田园风光转化为符号化的童趣世界，带来独特的怀旧与治愈感。 | 《星露谷物语》游戏田园混剪 / 像素动画短片《花园》 |
| 吉普力风格 | 以手绘水彩的细腻笔触见长，对自然环境的刻画层次丰富、栩栩如生。色彩饱满而柔和，光影带有梦幻的温暖感。尤其注重描绘风吹草动、云卷云舒等自然细节，赋予田园场景一种宁静而充满灵性的生命力。 | 《龙猫》（动画电影）、《魔女宅急便》 |
| 奇幻风格 | 充满神秘、奇幻和超自然元素，场景设计和角色造型都带有奇幻色彩，适合奇幻、科幻题材的视频。 | 《指环王》系列电影 |
| 水墨风格 | 中国传统的水墨画风格为蓝本，色彩以黑白色调为主，线条流畅，意境深远，适合表现中国传统文化题材的视频。 | 《小蝌蚪找妈妈》（水墨动画） |
| 蒸汽朋克风格 | 以19世纪的工业革命为背景，结合了蒸汽机和齿轮等元素，场景和角色造型都带有复古的工业风格。 | 《哈尔的移动城堡》（动画电影） |
| 田园风光风格 | 通常指一种通过视觉和听觉构建的理想化、治愈系的乡村或自然生活图景 | 《那人那山那狗》《普罗旺斯的夏天》 |
| 东方都市影视剧风格 | 职场、爱情、家庭、抗战、刑侦等体裁裁居多 | 《都挺好》《繁花》 |

### 2.2 常见色调类型及其特点

| 色调类型 | 色彩特征与光影特点 | 气质/情绪特点 |
|---------|-------------------|--------------|
| 暖色调 | 以黄、橙、红等暖色为主导，常降低色温。高光常渲染为金色或橙红色，阴影也带有暖色倾向，整体画面像被夕阳或炉火笼罩。 | 温暖、治愈、怀旧、安全。能激发食欲、归属感和幸福感，是"家"和"旧时光"的视觉象征。 |
| 冷色调 | 以蓝、青、冷绿为主导，常提高色温。高光偏冷白或淡蓝，阴影浓郁且带蓝/青调，整体通透或阴沉。 | 冷静、疏离、忧郁、神秘、科幻感。可表现孤独、理性、冬季清冷或雨夜的静谧。 |
| 黑白/单一色调 | 画面几乎为黑白或单一色调（如青橙、黄黑），仅保留一个主体或局部区域的鲜艳色彩（如一件红裙）。 | 戏剧性、焦点突出、象征意义。能瞬间引导观众视线，赋予特定元素强烈的情绪或象征意味。 |
| 蓝调 | 一种主导性的、浓郁的深蓝色笼罩整个画面。它不是简单的"偏蓝"，而是将大部分中间调和阴影区域染上深邃的蓝色，同时严格保留并提亮高光点（如灯光、月光、水面反光），形成强烈的冷暖对比。色彩饱和度中等偏高，但明度较低。 | 静谧、孤独、忧郁、神秘、内省、带有诗意的忧伤。它剥离了白天的喧嚣，赋予夜晚一种沉静而富有故事感的质感，比普通冷色调更具情绪张力和视觉风格。 |

## 3、`<Scene>` 标签涉及专业术语及样例

### 3.1 景别分类及判定方法

| 景别 | 定义 | 表达氛围 | 样例 |
|------|------|---------|------|
| 特写 | 肩部以上 | 拍摄局部细节，突出主题 | |
| 近景 | 胸部以上 | 强调表情或动作 | |
| 中景 | 膝盖以上 | 拍摄主体大部分，兼顾环境 | |
| 全景 | 人物从头到脚完整入镜 | 环境作为背景补充 | |
| 远景 | 人物全身可见，但环境仍占较大比例 | 强调大场景，人物较小，突出空间感 | |
| 空镜 | 没有任何人物出现，纯景色和场景拍摄 | 强调拍摄场景和渲染氛围 | |

### 3.2 拍摄角度分类及判定方法

| 拍摄角度 | 定义 | 样例 |
|---------|------|------|
| 仰拍 | 镜头从下向上拍摄 | |
| 俯拍 | 镜头从高处向下拍摄 | |
| 正面拍摄 | 镜头位于主体的正前方 | |
| 侧面拍摄 | 镜头位于主体的侧面 | |
| 背面拍摄 | 镜头位于主体的背后,只能看到主体的背影 | |
| 过肩镜头 | 从一个角色的肩膀后方拍摄对面的角色，是对话场景最常用的机位 | |

### 3.3 运镜分类及判定方法

| 运镜 | 定义 | 提示 |
|------|------|------|
| 固定镜头 | 摄像机的位置、拍摄角度都保持不变 | |
| 晃动镜头 | 相机在拍摄过程中位置和角度固定，画面有轻微晃动感 | 多为纪录片 |
| 手持静止镜头 | 画面固定且有轻微抖动，通过手持拍摄或后期模拟抖动，制造不稳定感 | 提示:跟固定镜头的区别在于，由于手持的不稳定性，画面会有微小的抖动和移动 |
| 推镜头 | 摄影机靠近主体,主体周围环境减少 | |
| 拉镜头 | 与推镜头相反,摄影机远离主体,主体周围环境增多 | |
| 移镜头 | 摄影机在**水平方向**上移动右移：相机在水平方向中按从左到右的轨迹进行移动拍摄左移：相机在水平方向中按从右到左的轨迹进行移动拍摄 | 提示：注意跟升降的区别 |
| 升镜头 | 指摄影机沿着**垂直方向向上**移动 | |
| 降镜头 | 与升镜头相反,摄影机沿着**垂直方向向下**移动 | |
| 环绕镜头 | 摄影机围绕主体或某个固定点旋转 | 提示：镜头围绕主体环绕拍摄，展示主体与环境的全方位关系。一般会有一个明确的主体作为拍摄对象，会围绕主体进行环绕，有明确的拍摄意图。 |
| 旋转镜头 | 摄像机沿镜头轴线旋转的拍摄方式，画面会产生倾斜或旋转效果 | |
| 摇镜头 | 摄影机为圆心，镜头左右或上下旋转，展示更大范围的上摇：相机从下往上转动拍摄，使画面中下方的景物逐渐移出视野，上方的景物进入视野下摇：相机从上往下转动拍摄，上方景物移出，下方景物进入画面右摇：相机位置固定，以自身为轴水平向右转动拍摄镜头左摇：相机位置固定，以自身为轴水平向左转动拍摄 | 提示：与环绕镜头有明显区别，环绕镜头一般是以被拍摄主体为圆心。 |
| 跟镜头 | 摄影机跟随主体运动 | 提示：一般来说，镜头属于推进或者拉远，跟平移的区别在于跟镜头一般会有明确的跟随主体，比如镜头跟着某个人或者动物一起前进后退，镜头会有明显位移。 |

### 3.4 常见拍摄手法及判定方法

| 拍摄手法 | 定义 |
|---------|------|
| 希区柯克变焦 | 推镜头+变焦拉（或拉镜头+变焦推）同时进行，主体大小不变但背景急剧变化，制造眩晕/不安感 |
| 延时摄影 | 以较低频率间隔拍摄（如每5秒拍1张），后期合成视频，将漫长过程压缩为短时间（如日出、花开）|
| 手持摄影 | 摄影师手持摄像机拍摄，画面带有自然晃动，增强真实感、临场感、纪实风格 |
| 慢动作 | 用高帧率（如120fps/240fps）拍摄，正常速度播放时动作变慢，展现细节、渲染情绪（如水滴、爆炸）|
| 快动作（加速） | 用低帧率拍摄或后期加速，压缩时间，表现忙碌、荒诞、喜剧效果 |
| 速度渐变（Speed Ramp） | 同一镜头中快慢速度交替变化，如从慢动作突然加速，节奏感极强，常用于MV和短视频 |

### 3.5 易混淆概念区分规则

| 易混淆项 | 正确标注依据 |
|---------|------------|
| 镜头拉近 vs 推镜 | 镜头拉近=仅**变焦**（相机不动）；推镜=物理移动相机 |
| 晃动 vs 抖动 | 晃动=无规律运镜；抖动=手持相机/车载相机无意识移动 |
