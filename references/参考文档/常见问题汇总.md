## 2、人物和场景问题

| 序号 | 出现的问题/规则要求 | 截图 | 错误解析 |
|------|-------------------|------|---------|
| 1 | 人物和场景需要仅针对当前**视频片段内**容进行描述 | | 红帽子可能是前/后一个片段出现的，非当前视频片段<br>健身房场景为下一个片段的内容 |
| 2 | 描述要用自然化语言，不能仅是各个元素简单堆砌 | | |
| 3 | 基本信息缺失——人物情绪表情、动作等基本信息缺失 | | |
| 4 | 基本信息缺失——人物年龄缺失 | | |
| 5 | 关键信息缺失，未描述到重点 | | |
| 6 | | | |
| 7 | 在描述具体物品时，应尽量明确主体（无论是人物还是物体） | | 使用"小物件"这种模糊的称呼来描述物品时，AI可能会生成不符合预期的结果。因此，在描述具体物品时，应尽量明确主体（无论是人物还是物体）。如果不确定具体物品，可以通过图搜工具进行查找。例如：三星的钥匙扣追踪器是一种具体的产品。尤其本视频中所谓的"小物件"恰恰是视频主角。 |
| 8 | 用词不当 | | |
| 9 | 语句不符合逻辑 | | 有"先"就证明有"后"，但实际该片段没有 |
| 10 | 非常规画面（脱离大众认知的场景）、特殊场景需在描述中有所体现，例如卡通效果、特效展示等 | | 其实还是描述准确性的问题 |
| 11 | 需要对片段内容动态描述，不能只描述静态画面 | | 例如：一个人走在街上，突然蹲下系鞋带，关键帧为系鞋带，不能只针对系鞋带的画面描述，要描述整个走动→蹲下→系鞋带的过程 |
| 12 | 客观事实陈述错误 | | 例如：主体之间的方位关系描述有误 |
| 13 | AI生成的结果看起来未做任何更改的 | | 正确描述：<br><br> |

## 3、运镜问题

| 序号 | 出现的问题/规则要求 | 截图 | 错误解析 |
|------|-------------------|------|---------|
| 1 | 定义不清晰——【跟镜头头】 | | 关于推镜头和跟镜头：需要理解拍摄意图，如果是明显跟随主体运动就写跟镜头头 |
| 2 | 定义不清晰——【降镜头】【降镜头】与【平移】混淆 | | 注意下不是下移，平移通常指水平方向左右移动 |
| 3 | 定义不清晰——【摇镜头】 | | "摇"不是指晃动的意思，而是以摄像机为圆心，镜头左右或上下旋转，展示更大范围的场景。 |
| 4 | 明显不是【固定镜头】但没有做修正 | | |
| 5 | 描述不能是各类运镜的罗列，而是需要体现镜头变化，可参考如下描述方式一：先xx运镜，再xx运镜<br>描述方式二：xx运镜结合xx运镜<br>描述方式三：xx运镜切至xx运镜以上均可，无固定格式要求 | | 各类运镜的罗列，完全不能体现镜头变化 |
